# Turning a LED on with a button

## Demo
[Watch Video](https://youtube.com/shorts/I7Meg9fw69s?si=bsDi5XqqGJmtqm70)

## Description
In this project i light a led using a button 

##  How It Works
- When the button gets pressed led turns on 
- When the button is released led turns off 

##  Components
- Elegoo Uno R3 
- Button
- LED
- Resistor ( 220ohm, 10kohm)
- Breadboard
- Wiers

## Reflection
- Beginner friendly project
- Teaches about pinMode, input and outputs 
- Basic circuit building and debugging
